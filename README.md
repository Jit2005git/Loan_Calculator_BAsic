# 💰 Loan Calculator Basic

A simple and responsive **Loan Calculator** built using **HTML, CSS, and JavaScript**. This application helps users calculate their monthly EMI (Equated Monthly Installment), total interest payable, and total repayment amount based on the loan details entered.

---

## 📌 Features

- 📊 Calculate Monthly EMI
- 💵 Calculate Total Interest Payable
- 💳 Calculate Total Repayment Amount
- ⚡ Instant calculation
- 📱 Responsive and user-friendly interface
- 🎨 Clean and modern design

## ⚙️ How It Works

1. Enter the **Loan Amount**.
2. Enter the **Annual Interest Rate**.
3. Enter the **Loan Tenure** (in years or months).
4. Click the **Calculate** button.
5. The application displays:
   - Monthly EMI
   - Total Interest
   - Total Payment

---

## 🧮 EMI Formula

The calculator uses the standard EMI formula:

\[
EMI = \frac{P \times R \times (1+R)^N}{(1+R)^N-1}
\]

Where:

- **P** = Principal Loan Amount
- **R** = Monthly Interest Rate
- **N** = Number of Monthly Installments
